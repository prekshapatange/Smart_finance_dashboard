# Smart Finance Dashboard

A full-stack personal finance management system that helps users track income, expenses, and financial activity in a unified dashboard.

## Features
- Track income, expenses, and recurring transactions
- Categorize and analyze spending
- Visual dashboards for insights
- Bulk transaction import
- Date-based filtering
- Secure authentication

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- APIs: REST

## Run Locally

### Backend
cd backend

npm install

npm run dev

### Frontend
cd client

npm install

npm run dev
